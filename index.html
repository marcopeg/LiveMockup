LiveMockup

Bring to life your UI Wireframes!

What does this plugin do?

LiveMockup create interactive wireframes and let you show your clients how your User Interface works with user interactions capabilities.

WIREFRAMES - the way you do it today:

During UI prototipation process you send your client a lot of images to display your UI and to let him imagine how it will works:

LiveMockup and Wireframes - stand up and run!

With LiveMockup you use images above (nothing else) applying some user actions to each image.
Only one image is visible so you obtain a visual effect of a working app.

You can say:
"when click next button go display the second image"

Example 03 - no way to write less code than this example!

If you set up an unordered list of wireframe steps with nothing else than images inside list items LiveMockup automagically creates a circular slideshow of slides allowing you to run you mockup! 

<!-- HTML: a simple unordered list of images -->
<ul class="livemockup">
	<li><img src="img/slideshow_wireframe_001b.png" alt="slideshow_wireframe_001b" width="800" height="496"></li>
	<li><img src="img/slideshow_wireframe_002b.png" alt="slideshow_wireframe_002b" width="800" height="496"></li>
	<li><img src="img/slideshow_wireframe_003b.png" alt="slideshow_wireframe_003b" width="800" height="496"></li>
</ul>

// on document ready:
$('.livemockup').liveMockup();
